## 🧩 How It Works

agent-recall stores useful context from your coding sessions. That can include:

- What you asked the AI to do
- Files you changed
- Decisions you made
- Open tasks
- Notes about each project

When you return later, the app can bring that context back. This helps the AI stay on track without asking the same questions again.
